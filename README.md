Project Overview
This project is a GUI-based ATM Machine application developed in C++ using Qt Creator.
It simulates the basic functionality of an Automated Teller Machine (ATM) and provides a user-friendly graphical interface for performing common banking operations.
The project is designed as a semester project to demonstrate the use of C++ programming, Object-Oriented Programming (OOP), and Qt GUI development.
